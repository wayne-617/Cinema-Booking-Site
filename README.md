# Cinema-Booking-Site
CSCI 4050

## To start backend:
- `cd demo`
- `docker compose up -d`
- `mvn spring-boot:run`

## Backend Swagger API
`http://localhost:9090/swagger-ui/index.html#/`

# Database
`docker exec -it cinema_app_mysql mysql -u user -p`
Password: `userpassword`

## To start frontend:
- `cd frontend`
- `npm install`
- `npm start`
